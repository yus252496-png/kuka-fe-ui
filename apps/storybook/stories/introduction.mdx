# @kuka-fe/ui

库卡机器人工业级 UI 组件库。

## 设计原则

- **工业赛博风格** — 深色主题、科技蓝主色、细边框玻璃态
- **TypeScript First** — 完整类型推导
- **主题系统** — 基于 CSS Variables 的设计令牌
- **无障碍** — 遵循 WAI-ARIA 标准

## 快速开始

```bash
pnpm add @kuka-fe/ui
```

```tsx
import { Button } from '@kuka-fe/ui'

function App() {
  return <Button variant="primary">Hello</Button>
}
```
